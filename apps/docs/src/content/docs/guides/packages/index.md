---
title: 패키지 가이드
---

프로젝트에 맞는 패키지를 빠르게 고를 수 있도록 역할별로 정리한 허브입니다.

- 먼저 읽기: [패키지 선택 가이드](/guides/package-selection/)
- 대부분의 사용자 시작점: [k-msg](/guides/packages/k-msg/)
- 저수준 커스터마이징이 필요하면: [@k-msg/core](/guides/packages/core/)

## 목록

- [@k-msg/analytics](/guides/packages/analytics/): 배달 추적 데이터를 기반으로 통계와 리포트를 만듭니다.
- [@k-msg/channel](/guides/packages/channel/): 프로바이더별 채널 lifecycle helper와 인메모리 toolkit helper를 제공합니다.
- [@k-msg/core](/guides/packages/core/): 저수준 타입, Result, 에러, 복원력 유틸을 제공합니다.
- [@k-msg/messaging](/guides/packages/messaging/): KMsg 뒤에서 라우팅, 큐, 배달 추적, 런타임 어댑터를 담당합니다.
- [@k-msg/provider](/guides/packages/provider/): 내장 provider 구현체와 onboarding 메타데이터를 제공합니다.
- [@k-msg/template](/guides/packages/template/): 템플릿 파싱, 치환, lifecycle, toolkit 유틸을 제공합니다.
- [@k-msg/webhook](/guides/packages/webhook/): 웹훅 runtime, persistence, 재시도, Cloudflare 어댑터를 제공합니다.
- [k-msg](/guides/packages/k-msg/): 대부분의 앱이 먼저 시작해야 하는 통합 facade입니다.
